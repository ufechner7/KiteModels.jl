Testing the develop branch before merging it

On AMD Ryzen 7 7840U.

Pass criteria for the performance:

First run:

  • simplifying the system < 45s

Second run:

  • system initialized < 15s
  • total time without plotting < 35s

Develop branch

juliaup default 1.11

cd ..

git push
